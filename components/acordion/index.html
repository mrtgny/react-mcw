Acordion properties

Props Type Description
data Array You can send your json-array data.
label String Label of acordion item.
open Boolean If true, acordion shows open.
child Function Function should return a component.
onClick Function You can set what does do after click the acordion item.
waveEffect Boolean If true, shows wave effect when clicked the list item.
light Boolean If waveEffect props is true, shows wave effect lighter when clicked the list item.
